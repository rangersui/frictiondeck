Five Rules. Nothing Else.
elastik is not a product. It's not a platform. It's a protocol — thin enough to fit beneath any framework, simple enough that any AI can write an implementation from scratch.
🔌
Listen
Accept connections on a port. HTTP. That's it. No WebSocket handshake, no gRPC, no protocol negotiation.
📨
Strings
Send and receive strings. No schema, no types, no serialization format. A string is a string is a string.
💾
SQLite
Store strings in one file. One table. Four columns. Every world is a row. Every row is a universe.
🔐
HMAC Chain
Sign every write. Chain-linked. Append-only. Immutable history. Not a database — a notary.

$ curl http://localhost:3005/home/work/read
